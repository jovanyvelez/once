# Tarea: Resumen de la Clase 8 de CS50x (HTML, CSS, JavaScript)

**Objetivo:**
Visualizar la [Clase 8 del curso CS50x 2026](https://www.youtube.com/watch?v=yYst7puZXjw) y redactar un resumen estructurado que demuestre la comprensión de la arquitectura web y sus lenguajes fundamentales.

## Instrucciones
Tu resumen debe estar organizado de forma lógica y es **obligatorio** que incluya y desarrolle los siguientes cinco elementos centrales abordados en la clase:

### 1. ¿Cómo funciona Internet?
Explica los fundamentos técnicos de la red. Asegúrate de abordar:
* El propósito de los protocolos **TCP/IP** (direccionamiento mediante IPs y entrega estructurada de paquetes).
* La función del sistema **DNS** (traducción de nombres de dominio legibles a direcciones IP).
* El rol del protocolo **DHCP** en la asignación automática de IPs a los dispositivos.

### 2. Protocolo HTTP y la Web
Detalla el lenguaje utilizado para solicitar y recibir páginas web. Debes incluir:
* Los componentes básicos de una URL (esquema, dominio, ruta).
* La diferencia fundamental entre los verbos HTTP (específicamente **GET** y **POST**).
* El significado y propósito de los códigos de estado más comunes vistos en clase (**200, 301, 404 y 500**).

### 3. HTML (Estructura)
Describe el propósito de HTML como el "esqueleto" de la página web. Tu sección debe cubrir:
* El uso general de las etiquetas y atributos esenciales (`<html>`, `<head>`, `<body>`, encabezados, párrafos, listas y tablas).
* Cómo se estructuran los **formularios** en HTML para capturar y enviar datos al servidor.

### 4. CSS (Estilo)
Explica cómo se maneja la estética y el diseño visual. Incluye en tu explicación:
* El funcionamiento básico de los **selectores** (cómo apuntar a elementos por etiqueta, por clase y por ID).
* El concepto de los frameworks de CSS, haciendo mención a la utilidad de herramientas como **Bootstrap** para acelerar el desarrollo frontend.

### 5. JavaScript (Interactividad)
Define el rol de JavaScript para dinamizar las páginas en el cliente. Asegúrate de explicar:
* Qué es el **DOM (Document Object Model)** y cómo permite a JavaScript modificar la página en tiempo real sin necesidad de recargarla.
* El manejo de **eventos** (cómo el código responde a las acciones del usuario, como clics o pulsaciones de teclado).
* Una breve mención a alguno de los ejemplos prácticos mostrados por el profesor (alertas, autocompletado, cambios de estilo en tiempo real o el uso de la API de geolocalización).
